How to add towns and items

  1. Open the data folder and duplicate one of the existing JSON files. Rename it, for example, frostmarket.json.
  2. Change the town value, the description, and the categories list in that file.
  3. Add or remove items inside the items array. Each item should have an id, name, price, category, and description.
  4. Open script.js and add your new town file to the townFiles list near the top.
  5. Commit the changed files and push them to GitHub. GitHub Pages will update the shop automatically.
{
  "town": "Frostmarket",
  "description": "A cold trade post known for hardy food and winter gear.",
  "categories": ["Common", "Local", "Special", "Winter Gear"],
  "items": [
    {
      "id": 1,
      "name": "Wool Travel Cloak",
      "price": 8,
      "category": "Winter Gear",
      "description": "A thick cloak lined for icy roads."
    }
  ]
}
